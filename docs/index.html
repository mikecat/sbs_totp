key

H(K XOR ipad, text)

Sbits = DT(HS)
Details
OffsetBits = low-order 4 bits of HS[19]
Offset = StToNum(OffsetBits)
P = HS[OffSet]...HS[OffSet+3]
Last 31 bits of P
Snum = StToNum(Sbits)
D = Snum mod 10^Digit
